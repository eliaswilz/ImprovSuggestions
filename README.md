# Improv Suggestions

A fast, focused iOS app for improv performers and comedy writers. Generate questions, word prompts, dialogue lines, and structured game setups on the fly.

## Modes

- **Questions** — Audience-style questions to inspire scene premises and opening offers.
- **Words** — Objects, locations, professions, and emotions as quick character and scene prompts.
- **Games** — Structured improv formats like First Line/Last Line and Changing Emotions.
- **Favorites** — Save and revisit the prompts you want to use again.
- **Settings** — Reset your library and view a quick how-to guide.

## Requirements

- iOS 17.0+
- Portrait orientation only on iPhone
